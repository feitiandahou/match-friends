# MatchFriends

一个基于 Vue + Spring Boot 的智能交友匹配系统。

## 功能特性

- 用户注册，登录

- 个人资料编辑
- 队伍编辑
- 队伍列表
- 队伍推送

```
## 🛠 技术栈
- **前端**: Vue 3, Pinia, Axios, Tailwind CSS
- **后端**: Spring Boot 3, JPA, MySQL,
```

## 技术栈

- **前端**： Vue3, pinia, Axios, Tailwinds Css
- **后端**：Spring Boot 3, Mysql, Redis
